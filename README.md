# JavaWeb 图书管理系统

## :smile:Introduction

基于JavaWeb的图书管理系统，主要用于图书信息的录入修改以及图书的借阅归还。</br>

相关技术：Java, JSP, Tomcat, MySql, Servlet, Html, CSS, JavaScript</br>



## :bulb:Information

运行环境：

- Tomcat 8.5.83

- MySQL 5.5.62

- JDK 1.8

  

## :triangular_flag_on_post:实现的功能

### 1.用户管理模块


- 登陆功能
- 验证码校验
- 安全退出
- 修改登陆密码

### 2.会员管理模块

- 会员开卡
- 注销会员
- 查看会员
- 修改会员

### 3.图书管理模块

- 对图书的增删查改
- 对图书类别的增删查改

### 4.常用功能模块

- 图书借阅
- 归还图书
- 历史借阅信息查询
